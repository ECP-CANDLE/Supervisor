
== Infer case

. Identify your output directories
. Make an infer UPF:
+
----
$ ./mk-infer-upf.sh upf-infer.txt <directories...>
----
+
This will find the files and paste together JSON document upf-infer.txt
. Run
+
----
$ ./upf-infer.sh
----
